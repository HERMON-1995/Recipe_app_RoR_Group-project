# 📖 [Recipe_app] <a name="about-project"></a>

**[Recipe_app]** is an app that keeps track of all your recipes, ingredients. It will allow you to save ingredients, keep track of what you have, create recipes, and generate a shopping list based on what you have and what you are missing from a recipe.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Install Ruby
- Install Rails
- Install Postgresql

### Setup

Clone this repository to your desired folder:

```sh
  cd Recipe_app_RoR_capstone
  git clone https://github.com/HERMON-1995/Recipe_app_RoR_capstone.git
```

### Install

Install this project with:

```sh
  cd Recipe_app_RoR_capstone
  bundle install
```
```sh
  gem install rspec
```
```sh
  gem install --no-document rubocop -v '>= 1.0, < 2.0' # https://docs.rubocop.org/en/stable/installation/
```
```sh
  npm install --save-dev stylelint@13.x stylelint-scss@3.x stylelint-config-standard@21.x stylelint-csstree-validator@1.x
```

### Usage

To run the project, execute the following command:

```sh
  rails db:create
```
```sh
  rails db:migrate
```
```sh
  rails server
```

Go to http://localhost:3000/api-docs to view swagger-ui

### Run tests

To run tests, run the following command:

```sh
  rubocop --color
```
```sh
  npx stylelint "**/*.{css,scss}"
```
```sh
  rspec
```
